# Features:

**AI-Powered Recommendation System**: A hybrid model combining collaborative filtering, content-based filtering, and context-aware recommendations to deliver personalized product suggestions based on user interactions and preferences.

**Gemini-Powered Recommendation Explanations**: Transparent AI feature using Gemini LLM to analyze user interactions and cart details, providing human-readable explanations of why specific products are suggested to users.

**Loyalty Program**: Functional reward system with daily check-in points, gift voucher redemptions, and seller-optional discounts for loyal customers to enhance engagement and encourage repeat purchases.

**User Interaction Tracking**: Comprehensive system that captures product views, clicks, session times, add-to-cart/remove-from-cart actions, purchase history, and search queries to better understand customer behavior.

**Personalized Discounts**: Event-based and behavior-triggered discounts, including cart abandonment prevention with optional email notifications to increase engagement and conversion rates.

**Mobile-Optimized Interface**: Responsive, fast-loading UI that ensures seamless functionality across all devices with effective API integration.

**Multi-Role System Architecture**: Separate interfaces and functionalities for customers, business owners, and platform administrators, each with tailored access levels, management tools, and insights.

**Comprehensive Analytics Dashboard**: Real-time visualization of KPIs, revenue tracking, conversion analysis, and seasonal trends for data-driven decision making by both administrators and business owners.

**Multilingual AI Chatbot & Visual Search**: Advanced chatbot that supports multiple languages and allows users to search products using images, powered by Gemini's multimodal capabilities.

**Natural Language & Visual RAG Search**: RAG-based search system that uses pgVector to store and compute cosine similarity between product descriptions and visual descriptions generated by Gemini.

**Cart Urgency Indicators**: Visual elements in the shopping cart that create a sense of urgency to encourage purchase completion.

**Trend-Based Mass Discounts**: Platform-wide discounts for trending products and seasonal events, powered by Google Trends API integration.

**GraphQL Implementation**: Fast and efficient data retrieval for guest users, with more sophisticated recommendation algorithms reserved for logged-in users.

**ASK AI Feature**: Pop-up interface allowing users to query the product catalog using natural language, leveraging a RAG system built on product descriptions and AI-generated visual descriptions.

**Seller Management Tools**: Comprehensive product listing, pricing management, and inventory control tools for business users.

**Business Intelligence Alerts**: Automated notification system that alerts sellers to seasonal trends and festive events via email, helping them optimize their product listings and pricing.

**Advanced Filtering & Sorting**: User-friendly filters for categories, price ranges, and other product attributes to enhance the shopping experience.

# Tech Stack:

- **Frontend**: Next.js
- **Database**: PostgreSQL, pgVector (for vector embeddings)
- **ORM**: Drizzle ORM
- **Backend Services**: Supabase, GraphQL
- **AI Integration**: Gemini SDK (multilingual multimodal model)
