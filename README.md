# SmartMART

SmartMart is an AI-powered e-commerce platform focused on delivering personalized shopping experiences using advanced recommendation systems, intelligent search, and real-time analytics.

## Features

### 1. <u>AI-Powered Recommendation System</u>
Hybrid recommendation engine combining collaborative filtering, content-based filtering, and context-aware signals to provide personalized product suggestions.

### 2. <u>Recommendation Explanations (Gemini)</u>
Generates human-readable explanations for recommendations based on user behavior, cart data, and interaction history.

### 3. <u>Loyalty Program</u>
Reward system with daily check-ins, points, voucher redemption, and seller-defined discounts to improve retention.

### 4. <u>User Interaction Tracking</u>
Tracks:
- Product views and clicks  
- Session duration  
- Cart actions  
- Purchase history  
- Search queries  

### 5. <u>Personalized Discounts</u>
Behavior-based discount system including:
- Cart abandonment triggers  
- Event-based offers  
- Optional notifications  

### 6. <u>Mobile-Optimized Interface</u>
Responsive UI with fast load times and efficient API integration.

### 7. <u>Multi-Role Architecture</u>
Separate systems for:
- Customers  
- Sellers  
- Administrators  

### 8. <u>Analytics Dashboard</u>
Provides:
- KPI tracking  
- Revenue insights  
- Conversion metrics  
- Seasonal trend analysis  

### 9. <u>Multilingual Chatbot & Visual Search</u>
Supports:
- Multi-language interaction  
- Image-based product search  

### 10. <u>Natural Language & RAG Search</u>
Uses pgVector to:
- Store embeddings  
- Perform similarity search  
- Enable semantic product discovery  

### 11. <u>Cart Optimization Features</u>
Includes urgency indicators to improve conversion rates.

### 12. <u>Trend-Based Discounts</u>
Platform-wide discounts based on seasonal trends and product demand.

### 13. <u>GraphQL Integration</u>
Efficient data fetching:
- Lightweight queries for guests  
- Advanced logic for authenticated users  

### 14. <u>ASK AI Feature</u>
Allows users to query product catalog using natural language via a RAG-based system.

### 15. <u>Seller Management Tools</u>
Includes:
- Product listing  
- Pricing management  
- Inventory control  

### 16. <u>Business Intelligence Alerts</u>
Automated alerts for:
- Seasonal demand  
- Market trends  
- Pricing opportunities  

### 17. <u>Advanced Filtering & Sorting</u>
Supports filtering by:
- Category  
- Price  
- Product attributes

## Tech Stack

- Frontend: Next.js  
- Backend: Supabase, GraphQL  
- Database: PostgreSQL  
- Vector Database: pgVector  
- ORM: Drizzle ORM  
- AI Integration: Gemini SDK  

---

## Overview

SmartMart focuses on combining personalization, explainability, and intelligent search to improve user experience and business decision-making in e-commerce systems.
